# Travel Website Frontend

React + Vite + TypeScript + TailwindCSS

## Setup
```bash
cd Travel-Website-FE
npm install
```

Create `.env` (optional):
```
VITE_API_BASE_URL=http://localhost:5000
```

## Run
```bash
npm run dev
```
Open `http://localhost:3000`.

Home page will call backend `/health` and display status.
